# Bangumi Annual Game Report 2025

A personalized game collection report generator for Bangumi users.

## How to Run

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start development server**:
    ```bash
    npm run dev
    ```

3.  **Open in browser**:
    Navigate to `http://localhost:5173/?user=linxyc29300` (replace with your username).

## Features

-   Fetches game collections from Bangumi.
-   Filters for games marked/updated in 2025.
-   Displays cover art, rating, and comment.
-   Responsive grid layout.
-   Dark mode support (system preference).

## Deployment

You can deploy this directly to Cloudflare Pages, Vercel, or Netlify.

### Build Command
```bash
npm run build
```

### Output Directory
`dist`
